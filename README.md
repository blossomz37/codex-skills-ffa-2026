# Codex Skills FFA 2026

A small collection of Codex skills and snippets for testing, documenting, and teaching AI-assisted writing workflows.

## Included Skills

- `new-app-discovery` — A workflow for safely testing how Codex can interact with local desktop apps, including UI behavior, file storage, and round-trip validation.
- `scrivener-agent` — A Scrivener-focused skill for working with `.scriv` projects through Computer Use plus package-level verification.
- `snippets` — Reusable prompt and workflow fragments.

## Purpose

This repo is designed for author-facing AI workflows, especially experiments that prove what an agent can safely do before trusting it with real writing projects.

## Usage

Copy a skill folder into your Codex skills directory, then restart or reload Codex so the skill can be discovered.

Each skill includes a `SKILL.md` file with trigger guidance, workflow steps, safety rules, and references.
